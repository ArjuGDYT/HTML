The parts needed for a PC:

  • PC Chasis/Case
  • Control Processing Unit (CPU)
  • Graphics Processing Unit (GPU)
  • Motherboard
  • Random Access Memory (RAM) /Memory
  • Storage (Hard Drive/Solid State Drive/M.2/etc)
  • Power Supply Unit (PSU)
  • System Cooling (Fans or Water Cooling Unit)
  • Peripherals
